# Featured Plugins List

This document lists the "Featured" plugins that will be highlighted in the LMAPP marketplace.
These plugins are maintained by the core team or trusted community members.

## 1. Code Analyzer (`official.analyzer`)
- **Description:** Static analysis and code quality metrics for Python projects.
- **Features:** Cyclomatic complexity, maintainability index, linting integration.
- **Status:** Beta (v0.2.0)

## 2. Git Flow Manager (`official.gitflow`)
- **Description:** Automates standard git workflows (feature branches, releases).
- **Features:** `git flow feature start`, `git flow release`, PR creation.
- **Status:** Stable (v1.0.0)

## 3. Document Chatbot (`official.docchat`)
- **Description:** RAG-based chat interface for local documents.
- **Features:** PDF/Markdown ingestion, semantic search, citation support.
- **Status:** Beta (v0.5.0)

## 4. Auto-Translator (`community.translator`)
- **Description:** Offline translation using local NLLB models.
- **Features:** 50+ languages, batch translation.
- **Status:** Alpha (v0.1.0)

## 5. Project Summarizer (`official.summarizer`)
- **Description:** Generates high-level summaries of entire project structures.
- **Features:** Tree view, README generation, dependency mapping.
- **Status:** Stable (v1.1.0)
